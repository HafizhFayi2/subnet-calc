Pengertian CIDR (Classless Inter-Domain Routing)

CIDR adalah metode pengalokasian alamat IP dan routing yang lebih fleksibel dibandingkan sistem klasik (Class A, B, C). CIDR memungkinkan pembagian ruang alamat IP yang lebih efisien dengan menggunakan notasi prefix (misal: /24) untuk menentukan jumlah bit yang digunakan sebagai network identifier.

Konsep Dasar CIDR

  • Network Prefix: Bagian dari alamat IP yang mengidentifikasi jaringan (misal: 192.168.1.0/24)
  • Subnet Mask: Angka 32-bit yang menandai bagian network dan host (misal: 255.255.255.0)
  • Wildcard Mask: Kebalikan dari subnet mask (misal: 0.0.0.255)
  • Alamat Jaringan: Alamat pertama dalam subnet
  • Alamat Broadcast: Alamat terakhir dalam subnet untuk komunikasi broadcast

Rumus Penting CIDR

Jumlah Subnet = 2(n)
Jumlah Host per Subnet = 2(32-n) - 2
n = jumlah bit network (prefix)

Fungsi Setiap Komponen

  • Subnet Mask: Menentukan bagian mana dari alamat IP yang merupakan network ID dan host ID
  • Wildcard Mask: Digunakan dalam Access Control List (ACL) untuk menentukan range alamat
  • Alamat Jaringan: Digunakan sebagai identifikasi jaringan dalam routing
  • Alamat Broadcast: Digunakan untuk mengirim paket ke semua perangkat dalam subnet

Pengertian VLSM (Variable Length Subnet Mask)

VLSM adalah teknik subnetting yang memungkinkan penggunaan subnet mask yang berbeda-beda dalam satu jaringan, sehingga pengalokasian alamat IP menjadi lebih efisien. Dengan VLSM, administrator jaringan dapat membuat subnet dengan ukuran yang berbeda sesuai kebutuhan setiap segmen jaringan.

Konsep Dasar VLSM

  • Host Dibutuhkan: Jumlah perangkat yang perlu dialokasikan IP dalam suatu subnet
  • Subnet Mask: Menentukan ukuran subnet berdasarkan kebutuhan host
  • Rentang IP: Alamat IP yang dapat digunakan oleh perangkat dalam subnet
  • Daftar Subnet: Kumpulan subnet dengan ukuran berbeda dalam satu jaringan utama

Proses VLSM

  1. Identifikasi kebutuhan host di setiap subnet
  2. Urutkan subnet dari kebutuhan host terbesar ke terkecil
  3. Tentukan subnet mask untuk setiap subnet berdasarkan kebutuhan
  4. Alokasikan subnet secara berurutan dalam blok jaringan utama

Fungsi Setiap Komponen

  • Host Dibutuhkan: Menentukan ukuran minimal subnet yang diperlukan
  • Subnet Mask: Menyesuaikan ukuran subnet dengan kebutuhan host
  • Rentang IP: Menunjukkan alamat yang dapat digunakan perangkat
  • Daftar Subnet: Memetakan seluruh subnet dalam jaringan secara efisien

Pengertian Fiber Optik dan Sistem Identifikasi

Fiber optik adalah media transmisi jaringan yang menggunakan serat kaca pada corenya dan plastik sebagai lapisannya untuk mentransmisikan data dalam bentuk cahaya. Untuk memudahkan identifikasi, setiap core dalam kabel fiber optik diberi warna berdasarkan standar internasional.

Jenis-Jenis Kabel Fiber Optik

  • Single Mode: Untuk jarak jauh (hingga 100 km), diameter core kecil (9μm)
  • Multi Mode: Untuk jarak menengah (hingga 2 km), diameter core lebih besar (50/62.5μm)
  • Simplex: Satu serat, satu arah transmisi
  • Duplex: Dua serat, transmisi dua arah

Sistem Identifikasi Core

Kabel fiber optik menggunakan sistem warna untuk mengidentifikasi core dan tube:

  1. Setiap core memiliki warna unik sesuai urutan standar
  2. Setiap tube (kelompok core) memiliki warna unik
  3. Identifikasi dilakukan dengan kombinasi warna core dan tube

Urutan Warna Standar

1. Biru, 2. Oranye, 3. Hijau, 4. Coklat, 5. Abu-abu, 6. Putih
7. Merah, 8. Hitam, 9. Kuning, 10. Ungu, 11. Pink, 12. Tosca (Aqua)

Kegunaan Identifikasi

  • Memudahkan penyambungan kabel di lapangan
  • Mencegah kesalahan dalam instalasi jaringan
  • Mempermudah pemeliharaan dan troubleshooting
  • Memastikan konsistensi dalam jaringan besar
